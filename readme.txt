== Frequently Asked Questions ==

= What is HSN and why do I need it? =

HSN (Harmonized System of Nomenclature) is an internationally standardized system for classifying traded products. In India, HSN codes are mandatory for GST compliance and help determine the correct tax rates for products.

= Is this plugin compatible with my theme? =

Yes! The plugin is designed to work with any WordPress theme. You can customize the display styling to match your theme's design perfectly.

= Can I import my existing HSN codes? =

Absolutely! The plugin supports CSV import functionality. You can import your existing HSN codes along with descriptions and GST rates in bulk.

= Does this plugin calculate GST automatically? =

Yes, the plugin automatically calculates GST based on the HSN codes and their associated tax rates. You can also set custom GST rates for specific products.

= Can I use this plugin for export businesses? =

Yes! The plugin supports HSN codes for all types of businesses including export/import. HSN codes are essential for international trade documentation.

= Is the plugin translation ready? =

Yes, the plugin is fully translation ready and includes proper internationalization support.

= Can I customize where HSN codes are displayed? =

Yes, you can choose to display HSN codes on product pages, shop pages, cart, checkout, and order details. Each location can be configured independently.

= What happens if I deactivate the plugin? =

Your HSN codes data is safely stored in the database. If you reactivate the plugin, all your data will be intact.

= Is there a premium version? =

This is a complete, full-featured plugin available for free. All features are included with no premium upgrades required.

= How do I get support? =

You can get support through the WordPress.org support forums, or visit our GitHub repository for technical issues and feature requests.
