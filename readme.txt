== Description ==

WooHSN is a comprehensive HSN (Harmonized System of Nomenclature) code management plugin for WooCommerce that simplifies GST compliance for Indian businesses. 

**Key Features:**

* **Easy HSN Code Assignment** - Add HSN codes to products effortlessly through an intuitive interface
* **Bulk Operations** - Assign HSN codes to multiple products at once using CSV import/export
* **Smart Suggestions** - Get intelligent HSN code suggestions based on product titles and descriptions
* **GST Rate Management** - Automatically calculate and display GST rates based on HSN codes
* **Frontend Display** - Show HSN codes on product pages, cart, and order details
* **Database Management** - Comprehensive HSN code database with descriptions and GST rates
* **Reports & Analytics** - Generate detailed reports for GST compliance and business insights
* **Customizable Display** - Style HSN code display to match your theme
* **Developer Friendly** - Hooks, filters, and shortcodes for custom implementations

**Perfect for:**
* E-commerce stores in India
* Businesses requiring GST compliance
* Stores with large product catalogs
* Developers building custom solutions

**HSN Code Features:**
* Pre-loaded database with common HSN codes
* Custom GST rate override for specific products
* Validation and format checking
* Search and filter functionality
* Export capabilities for accounting software

**Display Options:**
* Product pages
* Shop loops
* Cart and checkout
* Order details and invoices
* Custom positioning
* Styling customization

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/woohsn/` directory, or install through WordPress admin
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooHSN > Settings to configure display options
4. Start adding HSN codes to your products!

== Frequently Asked Questions ==

= What is an HSN code? =

HSN (Harmonized System of Nomenclature) is a standardized system of names and numbers to classify traded products. It's required for GST compliance in India.

= Do I need to manually add all HSN codes? =

No! WooHSN comes with a pre-loaded database of common HSN codes. You can also import your own via CSV or use the smart suggestion feature.

= Can I customize how HSN codes are displayed? =

Yes! You can customize colors, fonts, positioning, and format. You can also use shortcodes for custom placement.

= Does this work with all WooCommerce themes? =

Yes, WooHSN is designed to work with any properly coded WooCommerce theme. Display options can be customized to match your theme.

= Can I export HSN data for my accountant? =

Yes! WooHSN includes comprehensive export functionality for reports and data sharing.

= Is this plugin suitable for large catalogs? =

Absolutely! The plugin includes bulk operations, caching, and is optimized for stores with thousands of products.
